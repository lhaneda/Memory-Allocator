◆ print_memory()

void print_memory ( void  )

Prints out the current memory state, including both the regions and blocks. Entries are printed in order, so there is an implied link from the topmost entry to the next, and so on. The data is written into standard output stream.

See also
write_memory for more generic version.

◆ realloc()

void* realloc ( void *  ptr,
size_t  size 
)

Changes the size of the given allocated block to the new one. If the given block is NULL, equal to malloc_unsafe(size). If the new size is 0, equal to free_unsafe(ptr). Thread-safe.

See also
realloc_unsafe for for implementation of the reallocation itself.
Parameters
ptr- data pointer of the existing block.
size- new size for the block.
Returns
pointer to the first byte of data inside the resized segment.
