Detailed Description

Explores memory management at the C runtime level.

To use (one specific command): LD_PRELOAD=/allocator.so command ('command' will run with your allocator)

To use (all following commands): export LD_PRELOAD=/allocator.so (Everything after this point will use your custom allocator – be careful!)
